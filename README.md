# Bucklew v. Precythe Timeline
A clean, interactive timeline exploring a landmark U.S. Supreme Court case on the death penalty.

## Overview
This project presents the key events and legal significance of *Bucklew v. Precythe (2019)* through a chronological, visually driven web interface.

## Highlights
- 📅 Timeline-based storytelling  
- ⚖️ Focus on constitutional interpretation  
- 🎨 Minimal, modern design  
- 🧭 Clear structure for readability
